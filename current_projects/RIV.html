Project Leads: Jack Mueller and Neva Rognes

Project Members: Cameron Distinti, Pavit Saini, Simran Alur, Braden Masternak, Rayaan Padaria, Daeyeon Yang, and Arden Knee

Motivation:

Our goal is to create a website that allows users to plot hypothetical high-speed rail lines across North America and visualize their potential impacts, specifically in terms of ridership, and reductions in CO2 emissions. Overall, this is intended to serve as an advocacy tool for high speed rail.

Development Process:

Python and SQLite were the primary backend technologies used to compute gravity model calculations between city pairs and model ridership and emissions from those calculations. The frontend was primarily built using HTML and JavaScript to allow visualization and analysis of these hypothetical rail lines.
