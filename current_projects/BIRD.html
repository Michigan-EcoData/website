Motivation:

The BIRD project team is primarily focused on using the principles of machine learning to develop a model capable of identifying the species of a bird given an input image.

Development Process:

To do this, we have been working on developing a convolutional neural network with various parameters and machine learning applications to create a model capable of this goal. Our primary technology for achieving this has been the PyTorch framework, with recent research into transfer learning.
